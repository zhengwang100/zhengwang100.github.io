写给热爱“数据”研究的学生们

研究方向：“数据”是整个信息科学的核心基础，也是一门长期引领“新浪潮”的科研方向。据统计，数据库领域诞生了近50年来最多的四位图灵奖获得者，2022年全球的“独角兽”企业中有近一半是以“数据”相关技术作为其核心竞争力。

学生培养：实验室起源于清华大学，成长于交通大学，成员主要分布在上海、北京和港澳等地区，长期通过线下/线上交流合作。目前主要是交大的同学，也曾有清华NOIP保送的同学，和其他学校的ACM队的同学。交大的平台足够大、“数据”方向足够新浪潮，这里你的路会越走越宽。

方向选择：实验室从事最前沿的数据挖掘和（区块链）数据库方向的研究，学生可以根据兴趣自由选择，如下：

1) Data mining: social network analysis, graph neural networks, and deep learning. In this direction, we design new algorithms/theories for graph data related deep learning, which can be used in various data science scenarios (like Alibaba/Google/TikTok). Therefore, if you prefer mathematical theories (like matrix analysis and probability statistics), you are very encouraged to join the data mining group.

2) Database: blockchain related data architectures, and large-scale transaction processing systems. In this direction, we design new architectures/algorithms for the next-generation database systems, which can be used in various database scenarios (like Ali-Cloud/Amanzon AWS/Google-Cloud). Therefore, if you prefer data structures and algorithms (like binary-tree and dynamic programming), you are very encouraged to join the database group.
